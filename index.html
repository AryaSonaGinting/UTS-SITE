🔥 Apa itu JavaScript? 🔥

JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi interaktif dan dinamis. Kita bisa membuat animasi, validasi form, menampilkan pesan, atau bahkan game berbasis web.

🌿 Contoh Program Sederhana 🌿

<script>
  function halo() {
    alert("Halo Dunia!");
  }
</script>

<button onclick="halo()">Klik Aku</button>

🎶 Fungsi JavaScript 🎶

  • Menambah interaktivitas halaman web
  • Mengatur tampilan elemen secara dinamis
  • Mengambil dan memproses data dari server
  • Membuat aplikasi web dan game sederhana
