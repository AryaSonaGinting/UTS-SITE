🔥 Apa itu Tailwind CSS? 🔥

Tailwind CSS adalah framework utility-first yang membuat proses styling halaman web sangat cepat. Dengan Tailwind, kamu tidak perlu menulis CSS dari nol — cukup gunakan class siap pakai untuk warna, ukuran, layout, atau animasi.

🌿 Keunggulan Tailwind 🌿

  • ⚡ Cepat dalam pembuatan desain.
  • 🎨 Tampilan seragam tanpa file CSS tambahan.
  • 📱 Responsif dengan mudah menggunakan breakpoint siap pakai.

🎶 Contoh Penggunaan 🎶

Dengan Tailwind, kita bisa langsung menulis:

<button class="bg-green-500 text-white px-4 py-2 rounded-lg hover:bg-green-700">Klik Saya</button>
