🔥 Apa itu HTML? 🔥

HTML (HyperText Markup Language) adalah bahasa dasar dalam membangun struktur halaman web. HTML menggunakan tag atau elemen untuk menentukan bagaimana teks, gambar, tautan, dan elemen lain ditampilkan di halaman browser.

🌿 Struktur Dasar HTML 🌿

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
  </body>
</html>

🎶 Elemen Penting HTML 🎶

  • <h1>–<h6> → Judul
  • <p> → Paragraf
  • <a> → Tautan
  • <img> → Gambar
  • <table> → Tabel
